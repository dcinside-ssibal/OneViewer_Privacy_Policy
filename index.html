1. 서론

본 개인정보처리방침은 'OneViewer for dcinside' Chrome 확장 프로그램(이하 '확장 프로그램')이 사용자의 데이터를 어떻게 수집, 사용, 보호하는지에 대해 설명합니다. 저희는 사용자의 개인정보를 매우 중요하게 생각하며, 최소한의 정보만을 수집하여 서비스의 핵심 기능을 제공하는 데 사용합니다.

2. 수집하는 정보

확장 프로그램은 서비스 제공을 위해 다음과 같은 정보를 사용자의 컴퓨터에 저장합니다. 이 정보는 외부 서버로 절대 전송되지 않습니다.

  • 읽은 게시물 목록 (Viewed Post History)
    • 수집 항목: 사용자가 'OneViewer'를 통해 확인한 게시물의 URL 목록.
    • 저장 위치: 사용자의 브라우저 내 로컬 스토리지(localStorage). 이 데이터는 다른 기기와 동기화되지 않으며, 브라우저 캐시를 삭제하면 제거될 수 있습니다.
  • 확장 프로그램 설정 (Extension Settings)
    • 수집 항목: 확장 프로그램 활성화 여부(extensionEnabled), 다크 모드 사용 여부(darkMode)와 같은 설정값.
    • 저장 위치: Chrome 동기화 스토리지(chrome.storage.sync). 이 데이터는 사용자가 Chrome에 로그인한 경우, 동일한 Google 계정을 사용하는 다른 기기와 동기화될 수 있습니다.

3. 정보 사용 목적

수집된 정보는 다음의 목적을 위해서만 사용됩니다.

  • 읽은 게시물 목록: 사용자가 이미 읽은 게시물을 목록에서 시각적으로 구분(회색 처리)하여 편의성을 높이기 위해 사용됩니다.
  • 확장 프로그램 설정: 사용자가 설정한 옵션(예: 확장 기능 On/Off, 다크 모드)을 유지하여 일관된 사용 경험을 제공하기 위해 사용됩니다.
